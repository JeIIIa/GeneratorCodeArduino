# Генератор кода Arduino для датчиков
Графическое приложение для автоматической генерации кода Arduino для работы с аналоговыми датчиками.

## Особенности

- **Поддержка 4 типов датчиков**:
  - Потенциометр
  - Фоторезистор
  - Датчик наклона
  - Датчик вибрации
- **Удобный интерфейс** для добавления датчиков
- **Автоматическое распределение пинов** (A0-A5)
- **Генерация готового кода** с комментариями
- **Валидация входных данных** (проверка на дублирование пинов)

## Установка и запуск

### Требования
- Python 3.8 или выше
- PySide6
